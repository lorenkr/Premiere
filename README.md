# Avatar Movie Ticket App 

An Android application designed as a UI/UX prototype for selecting and purchasing tickets to the movie *Avatar*.  
This project focuses on screen design, interactions, and user flow — without backend implementation.

##  Features

- Display a brief summary of the movie  
- Choose ticket quantity  
- Select ticket type: **Adult / Child**  
- Pick screening **date** and **city**  
- Option to choose **3D** screening  
- Final order summary screen after confirmation  



##  Technologies

- **Android Studio**
- **Kotlin** / XML layouts
- Material Design components


##  Purpose of the Project

This app was built as part of a practice project focused on UI/UX, building user flows, and creating interactive screens in Android Studio.  
No backend or server communication is included.
